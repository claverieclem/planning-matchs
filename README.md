# Planning Matchs

Site React + Vite pour importer un fichier Excel, trier les matchs et générer un aperçu de planning.

## Lancer en local

```bash
npm install
npm run dev
```

## Mettre en ligne sur Vercel

1. Mettre tout ce dossier sur GitHub.
2. Connecter le dépôt à Vercel.
3. Vercel détecte automatiquement Vite.
